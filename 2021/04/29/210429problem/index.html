Apr 29, 2021 C++ Programming - Problem Report

2021年04月29日 921 字 大概 6 分钟

This C++ programming quiz is for the Freshmen student of Massey University at Apr 29, 2021.

Problem #a

Introduction

Write a C++ program that compares the average price of petrol in Auckland and Wellington. Read in a series of prices from various petrol stations in each region and then calculate the average petrol price for each region. Each petrol station must have a code to indicate which region it is in.

Notes

  • Your program must have two arrays – one array to hold Auckland data and one array to hold Wellington data.

  • The petrol stations can be entered in any order – do not assume that all stations in one region will be entered together.

  • The region code for Wellington is 4 and the region code for Auckland is 9

  • Use -1 to stop the data entry.

  • If no stations are entered for a region, then an appropriate message must be displayed.

  • Do not check for valid data – assume all data is entered correctly.

  • Make sure that you do not divide by zero.

  • The program does not need functions but you can use functions if you want to.

  • All prices must be displayed to 2 decimal places.

  • Your output must look the same as in the examples below.

Example

Example 1:

Enter code and petrol price for each region.
Wellington code is 4, Auckland code is 9, enter -1 to stop.
9 2.15
4 1.87
9 3.04
-1
Auckland average petrol price is $2.60
Wellington average petrol price is $1.87

Example 2:

Enter code and petrol price for each region.
Wellington code is 4, Auckland code is 9, enter -1 to stop.
9 2.46
9 2.57
-1
Auckland average petrol price is $2.51
No prices entered for Wellington.

Enter your C++ code in the text box below. If you like, you can test your program in Atom (or any other system) and then copy your program into the text box.

Answer

#include <iostream>
#include <cstdio>
using namespace std;
double Wellington_Petrol[5000], Auckland_Petrol[5000];

int main(int argc, char const *argv[])
{
    int zip_code;
    cout << "Enter code and petrol price for each region.\n";
    cout << "Wellington code is 4, Auckland code is 9, enter -1 to stop." << endl;
    cin >> zip_code;
    int well_num = 0, auck_num = 0;
    while(zip_code != -1)
    {
        if (zip_code == 9)
            cin >> Auckland_Petrol[auck_num ++];
        else 
            cin >> Wellington_Petrol[well_num ++];
        cin >> zip_code;
    }
    if (auck_num == 0)
        cout << "No prices entered for Auckland." << endl;
    else 
    {
        cout << "Auckland average petrol price is $";
        double sum = 0;
        for (int i = 0; i < auck_num; i ++)
            sum += Auckland_Petrol[i];
        sum /= auck_num;
        printf("%.2lf\n", sum);
    }
    if (well_num == 0)
        cout << "No prices entered for Wellington." << endl;
    else 
    {
        cout << "Wellington average petrol price is $";
        double sum = 0;
        for (int i = 0; i < well_num; i ++)
            sum += Wellington_Petrol[i];
        sum /= well_num;
        printf("%.2lf\n", sum);
    }

    return 0;
}
