Problem #a

Introduction

Write a C++ program that compares the average price of petrol in Auckland and Wellington. Read in a series of prices from various petrol stations in each region and then calculate the average petrol price for each region. Each petrol station must have a code to indicate which region it is in.

Notes

  1.  Your program must have two arrays – one array to hold Auckland data and one array to hold Wellington data.
  2.  The petrol stations can be entered in any order – do not assume that all stations in one region will be entered together.
  3.  The region code for Wellington is 4 and the region code for Auckland is 9
  4.  Use -1 to stop the data entry.
  5.  If no stations are entered for a region, then an appropriate message must be displayed.
  6.  Do not check for valid data – assume all data is entered correctly.
  7.  Make sure that you do not divide by zero.
  8.  The program does not need functions but you can use functions if you want to.
  9.  All prices must be displayed to 2 decimal places.
  10. Your output must look the same as in the examples below.
